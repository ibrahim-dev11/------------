# داشبۆردە - نۆتیفیکەیشن

## دستیابی بە داشبۆردە

1. بچۆ بۆ: **http://10.34.141.208:8000/admin**
2. لۆگین بکە بە هژماری بەڕێوەبەر
3. لە بازنەی سەریی، کلیکی لێ بکە: **ناردنی ئاگادارکردنەوە** (Notifications)

---

## 4 رێگا بۆ ناردنی نۆتیفیکەیشن

### 📣 بلاوکردنەوە (برودکاست)
- پێیام بۆ **هەموو بەکارهێنەرەکان** بنێرە
- بۆچوون: داگری نیگە
- لاببردن: ناوێـ ئاگادار + متن + زانیاری زیاتر (JSON)

**مثال:**
- ناونیشان: "بەڕێوەبەری سیستەم"
- پەیام: "سیستەم 2 سەعت بۆ پێداویستیمانی گێڕانەوە"

---

### 👤 یەک بەکارهێنەر
- پێیام بۆ **یەک بەکارهێنەری دیاریکراو** نێرە
- لاببردن:
  1. بەکارهێنەری دیاری بکە
  2. ناونیشان + پەیام + زانیاری

**مثال:**
- بەکارهێنەر: احمد محمد
- ناونیشان: "تۆی لە پێشتر کاری ماوە"
- پەیام: "مەستی خۆت بناسە، تۆ 3 داواکاری هەڵگیاندووە"

---

### 👥 گرووپی بەکارهێنەر
- پێیام بۆ **چندین بەکارهێنەری دیاریکراو** نێرە
- لاببردن:
  1. بەکارهێنەرەکان دیاری بکە (Multi-Select)
  2. ناونیشان + پەیام + زانیاری

**مثال:**
- بەکارهێنەران: 5-10 کەس
- ناونیشان: "هاوپێچ بۆ دانیشتنی کۆمێڵەی فێربوون"
- پەیام: "السبت ساعة 3 عصراً"

---

### 🏷️ بابەت (Topic)
- پێیام بۆ **گروپی بابەتی دیاریکراو** نێرە
- بۆچوون: سڵاکانی ئاگانەی تایبەت (بۆ نموونە: "هێڵی زمانی عربی")
- لاببردن:
  1. ناوی بابەت (مثلاً: "arabic_teachers" یاخود "دامەزراوەی کوردستان")
  2. ناونیشان + پەیام + زانیاری

**مثال:**
- ناوی بابەت: "announcements"
- ناونیشان: "ئاگادار"
- پەیام: "کۆبونی ئایندە بە زووی"

---

## 🔌 ناڕاستی بە یەک بەکارهێنەر

لە بەرگی **بەکارهێنەران** (Users):
1. بەکارهێنەرێک دۆزنہ
2. کلیکی **ناردنی نۆتیفیکەیشن** بکە
3. ناونیشان و پەیام بنووسە
4. **ناردن** کلیکی لێ بکە

---

## 📋 زانیاری زیاتر (JSON) - بۆچوون

لە هەموو تابەکاندا، دەتوانیت "زانیاری زیاتر" بنووسیت وەکو JSON:

```json
{
  "link": "/institutions/5",
  "type": "announcement",
  "priority": "high"
}
```

**مثال برای Filament:**
```json
{
  "action": "open_app",
  "target_page": "institutions"
}
```

---

## ⚠️ یادداشتی گرنگ

1. **FCM Token**: بەکارهێنەر دەبێ ئەپلیکەیشنەکە یاخود سەیری کردبێت تا FCM تۆکنیان درێجتەبی
2. **ئاگاداری چالاک**: بەکارهێنەر دەبێ `notifications_enabled = true` بێت
3. **اتصال انترنت**: بەکارهێنەرانی ئێوە دەبێ اتصالی انترنت ئاکتیف هەبێت
4. **Firebase**: پێویستە Firebase credentials ئیتثبەتی بێ

---

## 🔐 بەرگی دەستکاری بەکارهێنەر

لە بەرگی  `بەکارهێنەران`:

| فیلد | مەبەست |  
|------|--------|  
| `fcm_token` | تۆکنی Firebase (ئۆتۆماتیکی |  
| `notifications_enabled` | ئاگادار چالاک یاخود ناچالاک |
